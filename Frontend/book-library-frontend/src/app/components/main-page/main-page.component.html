About This Project

Book Library is a full-stack web application for book lovers, students, and administrators. It allows users to explore a rich collection of books, filter by genres, and manage the library with an intuitive interface.

Features

  • Browse books by genre, author, or title
  • Paginated and filterable book and genre lists
  • Admin dashboard for managing books and genres
  • User authentication and roles
  • Modern, responsive, and beautiful UI

Technologies Used

  • Angular & Angular Material (Frontend)
  • ASP.NET Core & Entity Framework (Backend)
  • SQL Server (Database)

How to Get Started

  1. Register for an account or log in
  2. Browse books and genres from the navigation bar
  3. Admins can access the dashboard for management features
